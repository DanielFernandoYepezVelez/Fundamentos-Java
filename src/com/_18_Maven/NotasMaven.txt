¿QUE ES MAVEN?
Es una herramienta de software que sirve para
la construcción y gestión de proyectos JAVA Y
fue creado en el año del 2002.

Nos permite crear proyectos de una forma
estandarizada y practica para trabajar con
JAVA.

NOTA => ---------------------------------------
Esta herramienta es muy similar al package.json
cuando se trabaja con proyectos de npm.
-----------------------------------------------

MVNREPOSITORY(Repositorio Central De Maven)
Es un repositorio que esta en la nube para
descargar las dependencias.

POM.xml
Es el archivo de configuración de maven
donde esta la versión que se utiliza,
las dependencias que necesita el
proyecto, entre otras caracteristicas.
------------------------------------------

COMANDOS MAVEN

PARA CREAR O GENERAR EL PROYECTO CON MAVEN:
mvn archetype:generate

ESTRUCTURA DE TU PROYECTO Y LO VA A DIFERENCIAR
CON OTROS PORYECTOS:
-DgroupId=com.primerproyectomaven.example

ES EL NOMBRE DEL COMPILADO(empaquetado al final .jar)
O DEL PROYECTO:
-DartifactId=my-app-simple

TIPO DE ARQUITECTURA QUE ESTOY CREANDO:
-DarchetypeArtifactId=maven-archetype-quickstart
-DarchetypeArtifactId=maven-archetype-webapp

VERSION:
-DarchetypeVersion=1.4
--------------------------------------

GENERAR EL COMPILADO(el jar al final) DE UN PROYECTO:
mvn compile => Este comando hace el compilado de ese proyecto
y nos entrega una carpeta target, ahi estan todos
nuestros archivos ejecutable .class.

PERO RECORDEMOS QUE EL jar ES EL EJECUTABLE DEL PROYECTO YA
QUE LO EMPAQUETA:
mvn package => Este comando empaqueta toda nuestra aplicación
o proyecto y genera el COMPILADO que hemos definido
inicialmente, que por defecto es un jar.

PARA EJECUTAR DICHO EJECUTABLE ES CON EL COMANDO:
java -cp my-app-1.0-SNAPSHOT.jar com.mycompany.app.App

App => es el nombre de la clase Main

Nuestra aplicación podemos montarla en un servidor
para que se ejecute.

PARA ELIMINAR LA CARPETA TARGET:
mvn clean.
